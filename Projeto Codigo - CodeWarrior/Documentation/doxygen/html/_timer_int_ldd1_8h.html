Detailed Description

This TimerInt component implements a periodic interrupt. When the component and its events are enabled, the "OnInterrupt" event is called periodically with the period that you specify. TimerInt supports also changing the period in runtime. This TimerInt component provides a high level API for unified hardware access to various timer devices using the TimerUnit component.
